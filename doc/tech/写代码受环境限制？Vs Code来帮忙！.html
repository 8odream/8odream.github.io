VScode（VisualStudio Code）是微软发布的Web应用IDE，可以辅助编程，可以打开在Github的远程仓库并支持编辑。支持强大多样的拓展。同时拥有浏览器版本［https://vscode.dev］，基本功能一致，但是不能打开本地文件夹，不支持部分拓展，但是不受设备限制，可以随时使用。同时也可以在GitHub的仓库界面点击键盘的“> .”按键来快速打开当前的仓库编辑界面。软件无广告，完全免费。注意如果要使用中文界面，需要到拓展商店（免费）里下载语言包（浏览器版本不支持）。

VScode对于刚刚入门使用Github的用户比较友好，可以以图形界面来辅助Git的基础操作（如push等）。它可以编辑几乎所有语言（如MarkDown，C，C++，Java，Python，HTML，JavaScript……也支持冷门的语言），拥有代码提示和Tab/Enter键自动补全，错误警报，单步调试，远程运行，变量记录，编译运行，界面预览等功能。浏览器版本不支持运行程序。对于一个没有个人电脑和硬盘的我（痛苦的往事）来说，使用浏览器版本自然是最优解。注意，如果打开浏览器的VScode后出现设备不支持的字样，多刷新几次就好（先禁用缓存）。如果无法加载Github文件（无法连接到），也可以尝试多次刷新。（国内环境嘛……）

这里我再推荐一个软件PortalApp，可以在U盘上安装Windows软件。如果你有移动硬盘，把VScode安装在硬盘里就好。
